# Excel

workbook: is an Excel file that all the information

- contain at least one worksheet

worksheet: contains the actual data and calculations

- organized into a collection of cells arranged in the form of a table

cells: store individual pieces of data or calculations

excel menu

- file menu: used to manipulate workbooks
- home menu: used to format the appearance of data in cells
- insert menu: allows you to place specialized elements into your workbooks
  - such as charts, pictures, and shapes
- page layout menu: used to format the appearance of worksheets
- formulas menu: aid in performing calculations using Excel's built-in functions
- data menu: used to aid in importing and working with large amounts of data
- review menu: help in sharing an Excel workbook
- View menu: used to change how a worksheet is displayed on the screen

pivot tables: used to organize and summarize large amounts of data. qualitative
data: used to classify or create categories in the data. quantitative data: are
numerical and are used in calculations

create a pivot table:

- select Insert menu, select Pivot Table

Sum – the values in each category will be added together (default option for
numbers)

Count – the number of different values will be displayed for each category
(default option for text)

Average – the values in each category will be averaged

Max – the largest value for each category will be displayed

Min - the smallest value for each category will be displayed

Product – the values in each category will be multiplied together

Count Numbers – the number of values in each category that are numbers will be
displayed

Standard Deviation – the sample standard deviation for each category will be
displayed

Standard Deviation Population - the population standard deviation for each
category will be displayed

Variance - the sample variance for each category will be displayed

Variance Population - the population variance for each category will be
displayed
